====================当前版本信息====================
当前版本：beta_V2.0

发布日期：2012-05-15

文件大小：59K


====================修改历史====================

V2.0  进行了以下几个方面的优化：
      1. 修改AGENT协议，提高效率；
      2. 兼容使用1.0 API的代码，只需替换库头文件重新编译即可；
      3. 解决了1.0 SDK网络通讯的一些BUG；
      4. 对于实时转发的数据也写容灾（1.0是不写容灾的），提升安全性；
      5. 落地标识已经不在 API控制，杜绝第三方写错的情况。

V1.2  优化了数据上报日志类型。

V1.1  优化了数据上报日志类型。

V1.0  CEE_C++_SDK  beta版第一版发布。


====================文件结构信息====================
inc文件夹: 头文件
	dcapi_cpp.h

lib文件夹：统一数据接入（CSEC,CBA及模调日志数据上报）库文件，提供了32位和64位的静态库和动态库
	libdcapi_cpp-32.a
	libdcapi_cpp-32.so
	libdcapi_cpp-64.a
	libdcapi_cpp-64.so
